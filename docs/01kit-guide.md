# 01Kit 安装和使用说明

01Kit 是一个本地优先的浏览器工作台。它可以开启专注计时、屏蔽容易分心的网站，并记录浏览时间分布；后续也会继续放入更多浏览器里的个人小工具。统计、站点列表和设置都保存在浏览器本地。

## 安装

Chrome 应用商店正在上架中，暂时可以使用离线安装包。

离线安装步骤：

1. 下载 `01kit-chrome.zip`。
2. 解压到本地文件夹。
3. 打开 `chrome://extensions`。
4. 开启“开发者模式”。
5. 点击“加载已解压的扩展程序”，选择刚才解压的目录。

## 开始专注

打开插件弹窗，选择 25、45、60 分钟，或输入自定义时长，然后点击“开始”。

专注开始后，01Kit 会按照当前规则屏蔽网站。需要临时处理事情时，可以暂停 1、5 或 15 分钟；事情处理完后再继续专注。

## 设置屏蔽规则

01Kit 支持两种模式：

- 黑名单：专注时屏蔽列表里的网站。
- 白名单：专注时只允许访问列表里的网站。

弹窗里可以快速添加常用域名。完整列表可以在设置页管理。

设置页还有“全局黑名单”。这里的网站无论是否正在专注，都会先拦截一次，适合放最容易顺手打开的网站。

## 查看时间统计

设置页可以查看今日时间线、今日分类统计、网站排行和最近 7 天专注记录。

时间线只记录当前获得系统焦点的 Chrome 窗口。系统空闲、Chrome 没有焦点、当前页不是网页地址时，都会显示为“未统计”。这样可以减少后台页面带来的误差。

今日时间线和今日分类统计使用同一天数据。网站排行可以切换今日、本周和本月。

## 导出和导入数据

设置页支持导出 CSV 和 JSON。

- CSV 适合查看网站排行。
- JSON 适合备份和恢复 01Kit 数据。

卸载插件会清掉本地数据。重装前，建议先导出 JSON。重新安装后，可以在设置页导入这个文件。

## 隐私说明

01Kit 不上传浏览记录、专注记录或站点配置。

插件需要全站点权限，是为了在本地识别当前域名、统计停留时间和执行屏蔽规则。它不读取页面正文、表单内容、Cookie、账号密码或个人通信内容，也不会把这些数据发送到服务器。

你可以随时在设置页清除本地统计数据。
